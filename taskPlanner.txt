------------------------------------------------------------
USER Routes
------------------------------------------------------------
GET  api/user/:id 
-user_id to be passed as request params

POST api/user
-name, email, password, contact, latitude, longitude, preferred_time, budget to be passed as request body

DELETE api/user/:id
-user_id passed as request params


------------------------------------------------------------
EVENT Routes
------------------------------------------------------------
GET api/event/all 
-get all the events present in the collection

GET api/event/filter
-get all the events for a particular user based on their preferred time, budget and location (within 10kms)
-user_id to be passed as request body

GET api/event/:id
-get event of this particular event id
-event it to be passed as request params

POST api/event
-post event
-title, description, type, entry_fee, user_id, contact, start_time, end_time, prize, latitude, longitude, start_date, end_date  to be passed as request body

DELETE api/event/:id
-delete event  and the forum and comments related to this event
-event it to be passed as request params

------------------------------------------------------------
COMMENT ROUTE
------------------------------------------------------------
GET api/event/forum/comment/:id
-get comment via comment id
-comment_id to be passed in params

POST api/event/forum/comment
-post comment 
-user_id, forum_id, content to be passed as request body

DELETE api/event/forum/comment/:id
-DELETE comment via comment id
-comment_id to be passed in params


------------------------------------------------------------
FORUM ROUTE
------------------------------------------------------------
GET /api/event/forum/:id
-get all the comments belonging to this forum
-pass forum_id as request params

DELETE /api/event/forum/:id
- delete the forum 
-pass forum_id as request params


------------------------------------------------------------
INTERESTED USER ROUTE
------------------------------------------------------------
POST /api/event/interesteduser
-add interested user to the event
-event_id , user_id to be passed as request body
